# Шахматы с ботом (Python + Pygame)

Локальная игра «Шахматы с ботом» с детерминированным движком (Minimax + Alpha-Beta), полной шахматной логикой, системой coins, профилем игрока и магазином.

## Запуск

```bash
python3 -m pip install pygame
python3 main.py
```

## Структура проекта

- `main.py` — запуск игры, главное меню, переходы между экранами.
- `board.py` — правила шахмат, генерация и проверка легальных ходов.
- `pieces.py` — описание фигур и базовые функции для движения.
- `bot_engine.py` — шахматный бот (Minimax + Alpha-Beta, глубина 1–5).
- `economy.py` — формулы наград и обновление статистики.
- `shop.py` — каталог магазина, покупка и применение предметов.
- `profile.py` — загрузка/сохранение профиля игрока в JSON.
- `ui.py` — отрисовка доски, фигур, меню и интерфейса на Pygame.
- `player_profile.json` — создается автоматически при первом запуске.

## Возможности

- Полные базовые правила шахмат:
  - все фигуры и ходы;
  - взятие, шах, мат, пат;
  - рокировка;
  - взятие на проходе;
  - превращение пешки;
  - запрет нелегальных ходов (король не может остаться под шахом).
- Бот-алгоритм без ML/API:
  - Minimax;
  - Alpha-Beta pruning;
  - оценка: материал, центр, безопасность короля, мобильность;
  - настраиваемая сложность (1–5).
- Экономика:
  - coins у игрока;
  - награды за победу/поражение/ничью;
  - учет сложности и длительности партии.
- Магазин:
  - доски, скины фигур, темы UI, разблокировка сложностей;
  - покупки сохраняются;
  - купленные предметы можно применять.
